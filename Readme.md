# :car: Evulution

ULaval's electric car sales and production system.

## Usage

Launch the application

```java
mvn clean package
        java-jar target/jersey-jetty.jar
```

Run the tests

````java
mvn test
````

Manual code formatting

````java
mvn formatter:format
````

Manual code format validation

````java
mvn formatter:validate
````
